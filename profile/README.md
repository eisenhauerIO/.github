## eisenhauerIO

We develop causal thinking and build tools to improve decisions at scale.

### Software

| Tool | Description |
|------|-------------|
| [Online Retail Simulator](https://github.com/eisenhauerIO/tools-catalog-generator) | Generates fully synthetic retail data for end-to-end testing of causal inference workflows with controlled treatment effects |
| [Impact Engine](https://github.com/eisenhauerIO/tools-impact-engine) | Implements causal inference methods for measuring treatment effects in business decision contexts |
| [Portfolio Allocation](https://github.com/eisenhauerIO/tools-portfolio-allocation) | Optimizes resource allocation across initiatives based on estimated impact and uncertainty |

### Courses

| Course | Description |
|--------|-------------|
| [Impact-Driven Business Decisions](https://eisenhauerio.github.io/courses-business-decisions/index.html) | Learn to design systems that measure impact, make decisions under uncertainty, and create scalable learning from decisions |
